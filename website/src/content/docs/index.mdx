## What is Kissaten AI?

Kissaten AI is a native macOS application for building machine learning pipelines visually. Drag and drop nodes, connect them together, and run training jobs—all without writing boilerplate code.

## Pipeline Flow

Kissaten AI uses a **node-based workflow**:

```
DataLoader → DataSplit → Trainer → Evaluator → ModelExporter
     ↓           ↓          ↓
   Data      Train/Test   Model    Metrics    Export
   File       Split      Training  & SHAP    .joblib/.onnx
```

1. **DataLoader** — Load CSV, JSON, Parquet, or TXT files
2. **DataSplit** — Split into train/test sets with optional stratification
3. **Trainer** — Train models (12 algorithms), tune hyperparameters, or load existing models
4. **Evaluator** — Auto-detect model type, compute metrics, generate explanations
5. **ModelExporter** — Save to joblib, pickle, or ONNX format
6. **Script** — Write custom Python code with Monaco editor

## Output Panel Tabs

The bottom panel provides 7 views:

| Tab | Purpose |
|-----|---------|
| **Logs** | Real-time execution output with syntax highlighting |
| **Data Profile** | Dataset statistics and column information |
| **Metrics** | Model performance with charts and confusion matrix |
| **Runs** | Run history with filtering and comparison |
| **Models** | Model registry with versioning and staging |
| **Trials** | Hyperparameter tuning results from Optuna |
| **Serving** | HTTP model server with request metrics |
