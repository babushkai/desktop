Kissaten AIは一般的な操作のキーボードショートカットをサポートしています。これらを覚えてワークフローを高速化しましょう。

## キャンバスナビゲーション

| ショートカット | 操作 |
|---------------|------|
| **スクロール**または**2本指ドラッグ** | キャンバスをパン |
| **ピンチ**または**Cmd + スクロール** | ズームイン/アウト |
| **Cmd + 0** | ズームを100%にリセット |
| **Cmd + 1** | 全ノードを画面に収める |

## ノード操作

| ショートカット | 操作 |
|---------------|------|
| **クリック** | ノードを選択 |
| **Cmd + クリック** | 選択に追加 |
| **選択ボックスをドラッグ** | 複数ノードを選択 |
| **Cmd + A** | 全ノードを選択 |
| **Delete**または**Backspace** | 選択したノードを削除 |
| **Cmd + D** | 選択したノードを複製 |
| **Escape** | 全選択解除 |

## パイプライン操作

| ショートカット | 操作 |
|---------------|------|
| **Cmd + Enter** | パイプラインを実行 |
| **Cmd + .** | 実行中のパイプラインを停止 |
| **Cmd + S** | パイプラインを保存 |
| **Cmd + O** | パイプラインを開く/読み込む |
| **Cmd + N** | 新規パイプライン |

## エディタショートカット（Scriptノード）

Scriptノードでコード編集時、標準のMonaco Editorショートカットが適用：

### 基本編集

| ショートカット | 操作 |
|---------------|------|
| **Cmd + Z** | 元に戻す |
| **Cmd + Shift + Z** | やり直し |
| **Cmd + X** | 行を切り取り |
| **Cmd + C** | コピー |
| **Cmd + V** | 貼り付け |
| **Cmd + D** | 次の出現を選択 |
| **Cmd + Shift + L** | 全ての出現を選択 |

### ナビゲーション

| ショートカット | 操作 |
|---------------|------|
| **Cmd + G** | 行へ移動 |
| **Cmd + P** | クイックオープン |
| **Cmd + Shift + O** | シンボルへ移動 |
| **Cmd + Home** | 先頭へ移動 |
| **Cmd + End** | 末尾へ移動 |
| **Cmd + ↑/↓** | スクロールアップ/ダウン |

### コード編集

| ショートカット | 操作 |
|---------------|------|
| **Tab** | インデント |
| **Shift + Tab** | インデント解除 |
| **Cmd + /** | 行コメント切り替え |
| **Cmd + Shift + /** | ブロックコメント切り替え |
| **Cmd + Shift + K** | 行を削除 |
| **Alt + ↑/↓** | 行を上/下に移動 |
| **Alt + Shift + ↑/↓** | 行を上/下に複製 |
| **Cmd + Enter** | 下に行を挿入 |
| **Cmd + Shift + Enter** | 上に行を挿入 |

### マルチカーソル

| ショートカット | 操作 |
|---------------|------|
| **Alt + クリック** | カーソルを追加 |
| **Cmd + Alt + ↑/↓** | 上/下にカーソルを追加 |
| **Cmd + D** | 次のマッチに選択を追加 |

## ウィンドウ操作

| ショートカット | 操作 |

## ヒント

<Aside type="tip">
**筋肉記憶:** まず基本—**Cmd + Enter**（実行）、**Cmd + S**（保存）、**Delete**（ノード削除）—から始めて、徐々に他を覚えましょう。
</Aside>

### 高速ワークフロー

1. **Cmd + N** — 新規パイプライン
2. ツールバーでノードを追加
3. プロパティパネルで設定
4. **Cmd + Enter** — 実行
5. **Cmd + S** — 満足したら保存

### 複数ノードの編集

1. **Cmd + A** — 全選択
2. ドラッグして位置変更
3. **Delete** — キャンバスをクリア
4. または**Cmd + クリック**で特定のノードを選択

## ショートカットのカスタマイズ

<Aside type="note">
カスタムキーボードショートカットは将来のリリースで対応予定です。現在、デフォルトのショートカットは変更できません。
</Aside>

## プラットフォーム注記

全てのショートカットはmacOSで**Cmd**キーを使用します。将来のWindows/Linuxリリースでは、**Cmd**は**Ctrl**に置き換えられます。

| macOS | Windows/Linux |
|-------|---------------|
| Cmd | Ctrl |
| Option | Alt |
| Delete | Backspace |
