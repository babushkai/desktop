## Kissaten AIとは

機械学習パイプラインをビジュアルに構築できるmacOSアプリケーションです。ノードをドラッグ＆ドロップで配置し、接続するだけで、コードを書かずにモデルの学習から評価まで行えます。

## パイプラインの流れ

Kissaten AIでは、ノードを左から右へ接続してパイプラインを構築します。

```
DataLoader → DataSplit → Trainer → Evaluator → ModelExporter
     ↓           ↓          ↓          ↓           ↓
   データ     訓練/テスト   モデル    メトリクス   エクスポート
   読み込み    分割        学習      & 説明     .joblib/.onnx
```

1. **DataLoader** — CSV、Parquetファイルを読み込み
2. **DataSplit** — 訓練データとテストデータに分割（層化サンプリング対応）
3. **Trainer** — 12種類のアルゴリズムから選択して学習、ハイパーパラメータ探索も可能
4. **Evaluator** — メトリクス計算、SHAP分析、説明生成
5. **ModelExporter** — joblib、ONNX形式で保存
6. **Script** — カスタムPythonコードを実行

## 対応モデル

| 分類 | 回帰 |
|------|------|
| ロジスティック回帰 | 線形回帰 |
| Random Forest Classifier | Random Forest Regressor |
| Gradient Boosting Classifier | Gradient Boosting Regressor |
| SVM (SVC) | SVM (SVR) |
| K近傍法 (KNN) | K近傍法 (KNN) |
| MLPニューラルネットワーク | MLPニューラルネットワーク |

## 出力パネル

画面下部には7つのタブがあります。

| タブ | 内容 |
|------|------|
| **Logs** | 実行ログとエラーメッセージ |
| **Data Profile** | データセットの統計情報 |
| **Metrics** | 精度グラフ、混同行列 |
| **Runs** | 実行履歴の一覧 |
| **Models** | 登録済みモデルの管理 |
| **Trials** | ハイパーパラメータ探索の結果 |
| **Serving** | HTTPサーバーの状態 |
